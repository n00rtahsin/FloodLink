⚙️
BRUSHLESS MOTOR
DJI-2212 920kV × 4

High-efficiency outrunner brushless motors providing the primary lift force. At 920kV, each motor spins at 920 RPM per volt applied, delivering stable thrust with low vibration.

920 kV 7.4-14.8V Max 2000g Thrust 3-Phase BLDC Outrunner
🔌
ELECTRONIC SPEED CTRL
30A ESC × 4

Receives PWM signal from Pixhawk and modulates current to each motor via 3-phase switching. The 30A rating provides adequate headroom for peak loads during aggressive maneuvers.

30A Continuous 40A Peak 2-4S LiPo BEC 5V/3A PWM Input
🌀
PROPELLERS
DJI 9405 × 4 (2CW+2CCW)

9.4-inch diameter, 4.5-inch pitch propellers. Counter-rotating pairs cancel torque reaction, eliminating yaw drift. Matched to the 920kV motors for maximum efficiency in the 3S voltage range.

9.4" Diameter 4.5" Pitch 2 CW + 2 CCW CF Reinforced
🦴
AIRFRAME
S500 Carbon Fiber Frame

Cross-shaped carbon fiber quadcopter frame. 500mm diagonal wheelbase offers excellent balance between stability and agility. Carbon construction provides high strength-to-weight ratio.

500mm Diagonal Carbon Fiber Shock-Absorbing Mounts Modular Arms
🔋
MAIN BATTERY
3S 4200mAh LiPo

3-cell lithium polymer battery delivering 11.1V nominal. The 4200mAh capacity provides adequate energy for multiple test flights. High C-rate ensures voltage stability under peak motor loads.

3S / 11.1V 4200 mAh 25C Discharge XT60 Connector ~200g Weight
⚡
POWER MODULE
90A Current Sensor

Passes main battery power to PDB while feeding regulated 5V to Pixhawk. Continuously monitors battery voltage and current draw, transmitting data to Pixhawk for battery failsafe triggers.

90A Max 2-6S Input 5V/3A BEC Out Voltage Sensor Current Sensor
🔆
POWER DISTRIBUTION
MATEK PDB-XT60

Central power distribution hub connecting battery to all four ESCs. Integrated voltage regulators provide clean 5V and 12V rails for peripheral electronics. XT60 input ensures secure high-current connection.

XT60 Input 4× ESC Outputs 5V/2A BEC 12V/0.5A BEC 9-18V Input
🛰️
GNSS MODULE
M10 GPS (u-blox M10)

Multi-constellation GNSS receiver tracking GPS, GLONASS, BeiDou, Galileo, and SBAS simultaneously. Tracks up to 32 satellites with 10Hz position update rate for precise waypoint navigation.

u-blox M10 GPS+GLONASS+BeiDou Galileo+SBAS ±2.5m CEP 10Hz Update 115200 baud UART Built-in Compass

SMART PAYLOAD BOX SYSTEM

💻
MICROCONTROLLER
Arduino Nano (ATmega328P)

Central processor of the Smart Payload Box. Reads all sensor data, executes safety logic, controls actuators, and communicates ARM/DISARM status to Pixhawk via MAVLink over UART.

ATmega328P 16 MHz 32KB Flash 2KB SRAM 14 GPIO 6 ADC UART/I²C/SPI
⚖️
LOAD CELL + ADC
10kg Cell + HX711 24-bit

Strain-gauge load cell converts weight to microvolts. HX711 24-bit ADC amplifies and digitizes the signal with programmable gain (32/64/128). Achieves 0.1129% MAPE across 50g–1000g range.

10kg Capacity 24-bit ADC Wheatstone Bridge ±0.345g Abs Error 2-Wire SPI
🔒
DOOR SENSOR
Hall Effect Sensor

Non-contact magnetic field sensor detects door open/closed state. Magnet mounted on door; sensor on frame. Provides clean digital HIGH/LOW output without mechanical contact — no wear, no false triggers.

Non-Contact Digital Output 5V Logic Magnetic Field
🔧
DOOR ACTUATOR
SG90 Micro Servo Motor

PWM-controlled servo manages physical door locking/unlocking. Rotates to 0° (closed/locked) or 90° (open/delivery). Feedback-controlled position ensures repeatable door action without stall.

4.8–6V Supply 180° Range PWM Control 1.8kg·cm Torque ~9g Weight
🔊
ALERT BUZZER
Piezo Buzzer 5V

Audible alert system activated during safety check failures. Signals overweight condition, door open state, or low battery. Allows ground operators to identify issues without visual interface.

5V Supply Low Current Digital Trigger Audible Alarm

❌ CURRENT — 3S 4200mAh LiPo
Generic 3S 4200mAh LiPo
Prototype-grade LiPo. Low energy density. 3S (11.1V) limits motor efficiency — motors run cooler but waste thrust potential. Rapid voltage sag under load reduces actual usable capacity to ~60%. Sensitive to cold weather (Bangladesh monsoon night temperatures). No smart BMS. No cell balancing during flight. Fire risk if punctured during hard landing. Swelling common after 50–80 cycles.
RAPID VOLTAGE SAG NO BMS PROTECTION LOW ENERGY DENSITY FIRE RISK

Tattu Plus 6S 22000mAh Smart LiPo
Tattu Plus series: integrated smart BMS. Reports accurate SOC (State of Charge), cycle count, health, temperature, and per-cell voltage directly to flight controller via UART/CAN. 22000mAh capacity provides 25–35 min flight at payload. 6S (22.2V) dramatically improves motor efficiency (40% less current for same thrust = less heat, longer life).

Ultra-low internal resistance. Stable under 50A+ continuous draw. 25C genuine discharge. Fireproof case. Smart chip for authentic SOC reporting.

No 3D point cloud — only scalar distance. Cannot classify obstacle type. Field-of-view is narrow (25°) — can miss obstacles offset from center line.

// PAYLOAD CONTROLLER
❌ CURRENT — Arduino Nano
Arduino Nano (ATmega328P)
16 MHz 8-bit AVR — adequate for current task but severely constrained for future expansion. No RTOS — all tasks run sequentially, causing timing jitter in MAVLink parsing. 2KB SRAM limits data logging. No WiFi/Bluetooth for diagnostics. Only one hardware UART — shared between debug and Pixhawk. No FPU — floating-point weight calculations are slow. No SD card built-in for logging. No OTA firmware updates. Prone to brownout during servo actuation spike.
8-BIT ONLY NO RTOS 2KB SRAM NO WIFI/BT 1 UART ONLY
BDT 400 (~USD 4)

ESP32-S3 + Raspberry Pi 5 (Companion)

Raspberry Pi 5 Companion: Handles computer vision from RealSense, MAVLink routing, mission logging to NVMe SSD, 4G LTE modem, and AI-based landing zone detection. 4-core 2.4 GHz ARM A76. Full Linux OS.
